# Market Opportunity Score(MIP Score)

## Objectif

Créer un indicateur synthétique permettant d'évaluer l'intérêt commercial d'un produit.

## Critères

- Tendances : 30%
- Marge estimée : 25%
- Concurrence : 20%
- Prix : 15%
- Popularité : 10%

## Evolution

Version 1 :
Score basé sur des règles.

Version 2 :
Ajout de modèles statistiques.

Version 3 :
Modèle prédictif.
