# Market Intelligence Platform

> A modern data engineering project for e-commerce market analysis.

## Vision

Market Intelligence Platform est une plateforme d'analyse de données permettant d'aider les vendeurs à prendre des décisions basées sur des données provenant de plusieurs marketplaces.

Le projet est développé dans un objectif d'apprentissage du Data Engineering, de l'Analytics Engineering et de la Business Intelligence.

---

## Objectifs

- Construire un pipeline de données moderne
- Développer des compétences en Data Engineering
- Concevoir une architecture évolutive
- Fournir des tableaux de bord décisionnels
- Appliquer les bonnes pratiques du développement logiciel
